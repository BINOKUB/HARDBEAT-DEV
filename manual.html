HARDBEAT PRO 64

OFFICIAL USER HANDBOOK (V1.0)

1. Architecture Système

Hardbeat Pro est une station de travail Techno basée sur une architecture 64 Pas (4 Mesures). Contrairement aux séquenceurs classiques limités à 16 temps, Hardbeat permet de composer des boucles longues et évolutives.

Navigation (Paging)

L'interface affiche 16 pas à la fois. Utilisez les flèches << et >> situées en haut de chaque section pour naviguer entre les mesures :

  • Page 1 : Pas 1 à 16
  • Page 2 : Pas 17 à 32
  • Page 3 : Pas 33 à 48
  • Page 4 : Pas 49 à 64

2. Drums & Polyrythmie

La section rythmique comporte 5 pistes : Kick, Snare, Hi-Hat Closed (HHC), Hi-Hat Open (HHO) et FM Perc.

Le moteur Polyrythmique

Chaque piste de batterie possède son propre slider de longueur (Loop Length). C'est l'arme secrète du Hardgroove.

  • Standard : Réglez sur 16 pour une boucle normale (4/4).
  • Décalé : Réglez sur 3, 5, 12... pour que l'instrument se décale par rapport au Kick à chaque tour.
  • Longue évolution : Réglez sur 64 pour écrire un pattern unique sur 4 mesures sans répétition.

3. Synthétiseurs de Fréquence

Deux synthétiseurs monophoniques sont disponibles pour sculpter vos textures.

SEQ 2 (Lead / Texture)

Le synthé principal (Cyan). Idéal pour les séquences hypnotiques et les leads.

SEQ 3 (Extension / Rumble)

Accessible via le bouton + EXTEND. Ce module supplémentaire (Violet) est conçu pour les basses fréquences et le renforcement du groove (Rumble).

Sliders de Fréquence (Hz)

Chaque pas possède un fader dédié (50Hz - 880Hz).
Astuce : Touchez un fader pour entendre la note (Preview) avant de placer votre pas sur la grille.

4. Accents & Dynamique

Pour éviter le son "robotique", Hardbeat Pro gère les accents (vélocité).

Accents Drums (Visibles)

La ligne du bas de chaque piste Drum (les petits carrés) active l'accent. Un pas accentué joue plus fort et avec plus d'impact.

Accents Synths (Fantômes)

Les synthétiseurs gèrent aussi les accents (Volume + Ouverture du Filtre). Ces accents ne sont pas visibles sur l'interface pour garder la clarté, mais ils sont programmables via le Preset Generator.

5. Outils Live & Export

Snapshots (Mémoire Fréquences)

Le module de droite permet de sauvegarder l'état des faders du Synth 2.

  1. Cliquez sur S (Le bouton clignote rouge).
  2. Cliquez sur un Slot (1-4) pour sauver.
  3. En jeu, cliquez sur le Slot pour rappeler instantanément la mélodie.

Export Audio (.WAV)

Cliquez sur REC pour armer l'enregistrement. Le bouton devient rouge. L'enregistrement capture la sortie Master exacte (y compris vos manipulations en direct). Cliquez à nouveau pour stopper et télécharger le fichier WAV Studio Quality.

6. Le Preset Generator

Pour composer des morceaux complexes, utilisez l'outil externe generator.html.

  • Remplissez les séquences Drum et Synth (Mode Humain : 1 à 64).
  • Définissez les longueurs de boucles pour la polyrythmie.
  • Définissez les fréquences précises.
  • Ajoutez les accents dans les champs dédiés (cases jaunes).

Le générateur crée un bloc de code à coller dans le fichier presets.js de l'application.
